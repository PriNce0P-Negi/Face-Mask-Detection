# Face Mask Detection System

## Overview
This project focuses on detecting whether a person is wearing a face mask in real time using deep learning techniques. It uses a convolutional neural network based on EfficientNet-B0 along with OpenCV for face detection and live video processing.

## Features
- Real-time face mask detection using a webcam  
- Deep learning model based on EfficientNet-B0  
- Face detection implemented using OpenCV  
- Lightweight and efficient for CPU-based systems  

## Technology Stack
- Python  
- PyTorch  
- OpenCV  
- NumPy  

## Project Structure
- `train_model.py` – Script used for training the model  
- `detect_mask.py` – Script for real-time mask detection  
- `utils.py` – Utility functions for preprocessing and transformations  

## Note
The dataset used for training is not included in this repository due to size limitations.
